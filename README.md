# Kanban Mastery

A Kanban board application built with .NET 8, developed as part of the
Relevate Engineering (.NET) Academy program.

## Project Structure

- `KanbanApi/` — ASP.NET Core Web API backend.

## Getting Started

```bash
cd KanbanApi
dotnet restore
dotnet run
```
